Latest Project

MERN Stack Journal Web App

I wanted to learn more about backend development using node.js, so I decided to do a MERN stack project. This project has allowed me to learn how to develop a Rest API and frontend with React. The backend is set up to follow REST API rules and handles all communication to the database from the frontend application. The project includes JWT authetication stored in https cookies. The frontend React app allows user to create an account, login, and record journal entries. The journal entry page allows users to save, delete, and update entries. Above the journal entry section, users can also select a date to view previous entries.
